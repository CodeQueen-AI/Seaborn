## üîπ What is Seaborn?

*Seaborn* is a powerful *Python data visualization library* built on top of *Matplotlib* It is mainly used for creating *statistical and attractive graphs* with very little code

### ‚ùì Why do we use Seaborn?

* To *understand data visually*
* To create *statistical plots easily*
* To write *less code* with *better-looking graphs*
* To visualize *numerical and categorical data*

### üéØ Where is Seaborn used?

* Data Analysis
* Data Science
* Machine Learning (EDA ‚Äì Exploratory Data Analysis)
* Reports and Dashboards


## üìå Seaborn Plots ‚Äì One Line Definitions

### 01. Scatter Plot : Shows the *relationship between two numerical variables*

### 02. Line Plotn : Shows *trends or changes over time or sequence*

### 03. Bar Plot : Used to *compare values or averages across categories*

### 04. Density Plot : Shows the *smooth distribution curve of data*

### 05. Box Plot : Displays *data spread, median, quartiles, and outliers*

### 06. HeatMap : Represents data using *colors to show intensity or magnitude*

### 07. Pair Plot : Shows *pairwise relationships between multiple variables*

### 08. Violin Plot : A *combination of box plot and density plot*

### 09. Count Plot : Shows the *frequency (count) of each category*

### 10. Histplot : Displays the *frequency distribution using bins*

### 11. Displot : A flexible function to draw *distribution plots* (histogram, KDE, ECDF)

### 12. Pair Grid : An *advanced and customizable version of pair plot*

### 13. Facet Grid : Creates plots in *rows and columns based on categorical variables*

### 14. Reg Plot : Draws a *scatter plot with a regression line*

### 15. Implot : Used for *categorical scatter and regression plots*

### 16. Cat Plot : A *general function* for categorical plots like bar, box, violin, strip, etc

### 17. Styling & Theme : Controls the *appearance, colors, and background* of plots

### 18. Annotations & Highlights : Used to *add text, arrows, and highlight important points* on graphs
